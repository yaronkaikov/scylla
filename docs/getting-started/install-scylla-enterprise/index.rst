Keep your versions up-to-date. The two latest versions are supported. Also, always install the latest patches for your version.

Cloud Deployment
---------------------

To deploy ScyllaDB Enterprise on AWS, GCP, or Azure, go to 
`ScyllaDB Download Center <https://www.scylladb.com/download/>`_. 
Select your cloud provider and the ScyllaDB Enterprise version you want to 
deploy and follow the instructions.


Installation on Linux
--------------------------

You can install ScyllaDB Enterprise with 
:doc:`Web Installer </getting-started/installation-common/scylla-web-installer>`,
a platform-agnostic installation script (recommended). 

Alternatively, you can install ScyllaDB Enterprise using Linux packages. Go to 
`ScyllaDB Download Center <https://www.scylladb.com/download/>`_, select your 
platform and the ScyllaDB Enterprise version you want to install, and follow 
the instructions.

* :doc:`Install ScyllaDB with Web Installer (recommended) </getting-started/installation-common/scylla-web-installer>`
* :doc:`Install ScyllaDB Without root Privileges </getting-started/installation-common/unified-installer>`
* :doc:`Air-gapped Server Installation </getting-started/installation-common/air-gapped-install>`
* :doc:`ScyllaDB Developer Mode </getting-started/installation-common/dev-mod>`
